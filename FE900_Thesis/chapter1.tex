Page numbers in a thesis must be formatted as they are in this document.
To do this in Word:
\begin{itemize}
    \item Go to the Insert tab
    \item Select Page Number
\end{itemize}

There is no required citation style for references or a bibliography in a thesis.  If your advisor or department has a required or suggested citation style, that should be what is used in the thesis.  Frequently used citation styles are MLA\cite{gibaldi_mla}, APA style\cite{APA}, Chicago, ASM and AIP.

The suggested typeface for theses is 10 to 12 point Arial. Other suggested fonts are Times New Roman and Helvetica. Script and italic fonts should not be used except as needed in the body of the paper. Consistency should be maintained throughout the paper, using the same font for all text, diagrams, etc. on all pages.

Electronic material, such as CDs and multimedia can accompany the thesis. Each copy should be accompanied with a labeled CD. When bound, these are placed in an envelope in the back of the thesis.  Oversized materials (pages greater than 8$''$ x 11$''$) can be included in the paper if necessary. Oversized pages should be folded so that 1$''$ on the left hand binding edge remains, allowing the page to be opened once the paper is bound.
